# ChatBot
 java ChatBot
# Задачи
1) Создание начального набора команд для бота:
    echo - печать вашего ввода
    authors - имена авторов
    games - игровое меню
    date - печать текущей даты
    run - запуск указанного приложения
2) Создание возможности работы с 3D сценой, путем добавления или удаления уже добавленных
геометрицеских объектов, их вращения, изменения цвета, размера, позиции на сцене.

3) Добавление возможности загружать 3D объекты в различных форматах (STL, OBJ),
посредством парсинга соответствующих файлов, а так же возможности сохранять
всю сцену в один файл (сцена включает в себя все добавленные или загруженные модели)
в одном из поддерживаемых форматов.
